编程步骤

服务端

  1. 加载套接字(WSAStartup())
  2. 绑定套接字到IP和端口上(bind())
  3. 设置为监听模式(listen())
  4. 接收到请求后，返回本次请求所对应的套接字(accept())
  5. 发送与接受数据(send()/recv())
  6. 关闭套接字(WSACleanup())

客户端

  1. 加载套接字(WSAStartup())
  2. 发出连接请求(connect())
  3. 发送与接受数据(send()/recv())
  4. 关闭套接字(WSACleanup())
