web是什么？

  • World Wide Web 万维网
  • 非常普遍的互联网应用
  • 平时流量的网页
  • 经历了Web1.0，Web2.0等阶段
    • Web1.0主要是静态页面
    • Web2.0类似于博客等页面

web的工作流程

  • 通过dns请求，发送域名到dns服务器，得到解析后的ip地址，然后便可以通过解析完毕的ip地址来访问一个网址。

url的详细格式

  • schema://host[:port#]/path/…/[?query-string][#anchor]
    • schema：底层协议，如:http,https,ftp
    • host：服务器域名或者IP地址
    • port： 端口，默认80端口
    • path： 访问资源的路径
    • ?query-string： 发送给http服务器的数据
    • anchor： 锚

什么是HTTP?

  • web使用的协议，即超文本传输协议
  • 应用最广泛，web基础
  • Hyper Text Transfer Protocol

    什么是HTTPS?

  • 可以简单来说，HTTPS = HTTP + SSL
  • 相对于HTTP来说，HTTPS更为安全
  • HTTP标准端口是80，HTTPS标准端口则为443
  • HTTP无需加密，HTTPS对传输的数据进行加密
  • HTTP无需证书，HTTPS需要认证证书

Javascript Bom

  • 警告弹窗 alert()
  • 确认弹窗 confirm()
  • 提示弹窗 prompt()
    • Bom浏览器对象模型
    • 以后测试可能会用到
    • 检验是否有漏洞
    • 建议熟记

  • linux系统下，hosts文件则在 /etc/hosts
